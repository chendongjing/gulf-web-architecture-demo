******************部署说明*********************
1.建好mysql数据库，自己自定义
2.执行初始化数据脚本 gulf-web.sql
3.配置你的数据库链接
4.hbase部分可去掉，如果你不需要hbase，否则会报错

特别说明：
a.本系统使用mysql和Hbase数据库，Hbase的脚本暂不提供，不影响开发和拓展。你也可以把hbase部分去掉。
b.maven依赖的包我已放在这个文件夹下/gulf-web/src/main/webapp/WEB-INF/readme/maven仓库(后面发现文件太大，不放了要的直接Q我)
如果不想直接从网络下载，直接复制过去覆盖你的maven
仓库就可以了。
